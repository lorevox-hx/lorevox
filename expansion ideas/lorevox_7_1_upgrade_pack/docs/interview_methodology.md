
# Lorevox 7.1 Interview Methodology

## Core Model
Two-pass system:
- Pass 1: Vocabulary + identity
- Pass 2A: Timeline spine (DOB → present)
- Pass 2B: Narrative depth (scenes, meaning, relationships)

## Opening (Required)
1. When were you born?
2. Where were you born?
3. Thank you — this helps me build your timeline.
4. Are you ready to begin?

## Dementia-Safe Principles
- Never ask: "Do you remember..."
- Use recognition over recall
- Accept repetition
- Do not correct or challenge memory

## Prompt Strategies
- Favorites (music, food, places)
- Family anchors
- Sensory prompts
- Shared experiences

## Core Rule
Recognition > Recall
